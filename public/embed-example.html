aidekin embed example

This is a plain web page that embeds the aidekin widget with a single script tag. Look for the launcher in the bottom-right corner and click it.

Build first, then serve the output: npm run build && npm run preview, then open /embed-example.html on the preview server.

The only thing a site owner adds:

<script
  src="/loader.js"
  data-mode="text"
  data-title="aidekin"
  data-greeting="Hi! Ask me anything."
  data-accent="#29a383"></script>
